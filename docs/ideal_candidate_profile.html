Core Competencies Required

Required Monday.com Administration

Must configure automations to block FULFILLED without QA DOCS, enforce owner assignment, and build compliance dashboards.

Required Process Enforcement Mindset

Comfortable being the "gatekeeper" who holds the line on compliance. Won't back down when team pushes back on new rules.

Required Data Analysis

Can run queries, identify non-compliant items, track improvement metrics, and present findings to leadership.

Preferred Change Management

Will face initial resistance. Needs skills to communicate "why" and get buy-in without alienating the team.

Preferred Manufacturing/QA Background

Understands production workflows, spec sheets, batch records, COAs. Speaks the language of the floor.

Nice to Have Training/Documentation Skills

Can create Loom videos, write clear procedures, and run training sessions for the production team.
