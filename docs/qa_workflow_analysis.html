Recommended Actions

1. Add QA Inspection Template Link

  • Create new link column on Production board
  • Or add to Finalization board and mirror
  • Link to "PURE EARTH LABS Spec Sheet & Inspection Report" template

2. Enforce QA DOCS Upload

  • Add automation: Can't move to FULFILLED unless QA DOCS has file
  • Training: Step 13 is non-negotiable
  • Weekly audit of QA status items

3. Populate Spec Sheets

  • Backfill existing items with spec sheet links
  • Create spec sheets for Earth Harbor products
  • Consider mirroring from Finalization board

4. Link Schedule Sheets

  • Add Sprint/Bulk weekly sheet link to Schedule Sheet column
  • Or create board-level link accessible from any item

5. Pre-Run Checklist (Optional)

  • Create subitem template for pre-run verification
  • Steps 1-6 as checkable subitems
  • Auto-create when item enters "PRODUCING" status

6. Create QA Dashboard View

  • Filter: STATUS = "QA"
  • Show: Name, Spec Sheet, QA DOCS, Days in QA
  • Highlight items missing documentation
