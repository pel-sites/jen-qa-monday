Recommendations

1. Block FULFILLED Without QA DOCS

  • Add Monday.com automation
  • Prevent status change to FULFILLED if file_mktcs58s is empty
  • Show warning message with link to QA template

2. Require Owner Assignment

  • Block items from leaving BACKLOG without owner
  • Auto-assign based on LINE or product type
  • Weekly report of unassigned items

3. Add QA Template Link

  • Create new link column for QA Inspection Report template
  • Pre-populate with Google Drive link
  • Make visible in QA status view

4. Create QA Dashboard

  • View: STATUS = FULFILLED AND QA DOCS empty
  • Show items needing backfill
  • Track completion over time

5. Backfill Historical Data

  • Identify items with Google Drive docs
  • Bulk upload QA reports to Monday.com
  • Link existing spec sheets

6. Weekly Audit Report

  • Auto-generate incomplete item list
  • Send to QA Manager (Jen)
  • Track improvement metrics
